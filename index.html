Welcome to MadBomber.NET

Work-in-progress Website for the Online Multiplayer Action Maze Game MadBomber.

The game is currently being tested within a small group of testers. There are no public builds yet. Since development and design is done by a single person, progress is going to be slow and updates can be fairly infrequent. Public Builds will be released when the game is ready for it.

For now, the only thing that can be found on this Website is the public List of all currently running Gameservers. This will eventually be expanded with the Repository, which will be a place for User-created Maps and Game Types. Submission will not be handled through this Website though, this is simply a read-only frontend for the MadBomber API.

If you would like to help out with Playtesting MadBomber, feel free to join the MadBomber Discord. Simply follow the steps in the #getting_started channel to get access to the current development builds.
